TALLER DE ALGORITMO 2 - EJERCICIO RESUELTO

FUENTE: GOOGLE CODE JAM

El alfabeto latino tiene 26 caracteres y los teléfonos sólo tienen diez digitos en el teclado.Nos gustaría que sea más facil escribir un mensaje a su amigo ulizando una secuencia de pulsaciones de teclas para indicar los caracteres deseados. Las letras se asignan a los digitos como se muestran acotinuación: Para insertar el caracter “b”, por ejemplo, el programa debe presionar 22. Para insertar dos caracteres en secuencia de la misma tecla, el usuario debe hacer una pausa antes de pulsar la tecla para indicar una pausa. Por ejemplo, 2 2 indica “aa” mientras que le 22 indica “b”.
